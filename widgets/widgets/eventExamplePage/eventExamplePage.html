Button state is {buttonState}

The widget "ToggleButton" with name "my-button" below will emit event with topic "change.state" every time it become pressed on unpressed.

Page widget is subscribed to the child widget's event "change.state" and modifies "buttonState" context variable. That leads to changing badge in the text above by the page widget's behaviour class.
